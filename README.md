## 📖 Overview

This project demonstrates an **end-to-end Machine Learning workflow** for predicting house prices in India. It covers the complete pipeline—from data preprocessing and exploratory data analysis (EDA) to feature engineering, model training, evaluation, and deployment using Streamlit.

The goal is to provide users with an intuitive web application where they can input property details and receive an instant estimated house price.
